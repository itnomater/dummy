Script http-request

Description

Send a HTTP request.

Arguments

  1. String Entrypoint URL address.
  2. String Data to send in format: key=value.

In background it uses curl command. The real name of this script is _http-request.sh, but it uses the symlinks. When you run it for the first time, it creates 8 symlinks for itself named:

  • http-post.sh - Send a HTTP request using the POST method.
  • http-get.sh - Send a HTTP request using the GET method.
  • http-put.sh - Send a HTTP request using the PUT method.
  • http-delete.sh - Send a HTTP request using the DELETE method.
  • http-json-post.sh - Send a JSON HTTP request using the POST method.
  • http-json-get.sh - Send a JSON HTTP request using the GET method.
  • http-json-put.sh - Send a JSON HTTP request using the PUT method.
  • http-json-delete.sh - Send a JSON HTTP request using the DELETE method.

First four symlinks are used for sending the standard HTTP requests, the remaining four used for JSON requests.

Examples

# A HTTP request using the method PUT.
$ http-put.sh http://domain.com/entrypoint foo='value with spaces' bar='value "with" quotes'

# It will produce:
curl --request PUT 
     --cookie /tmp/cookie 
     --cookie-jar /tmp/cookie 
     --url http://domain.com/entrypoint
     -F 'foo=value with spaces'
     -F 'bar=value "with" qutoes'

# A JSON HTTP request using the POST method.
$ http-json-post.sh http://domain.com/entrypoint 'foo=value with spaces' 'bar=value "with" spaces'
 
# It will produce: 
curl --request POST 
     --header 'Content-Type: application/json'
     --cookie /tmp/cookie 
     --cookie-jar /tmp/cookie 
     --url http://domain.com/entrypoint 
     --data '{"value": "value with spaces", "bar": "value \" with \" qutoes"}'

You can use a verbose mode to see what exactly the request is. All you need is set global variable DEBUG=1 before run this script.

# For one execution:
$ DEBUG=1 http-json-post.sh http://domain.com/entrypoint foo=value bar=value

# For all executions in the current terminal.
$ export DEBUG=1 
$ http-json-post.sh http://domain.com/entrypoint foo=value bar=value
