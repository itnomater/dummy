Script test-directory

Description

Monitoring directory changes.

Arguments

  1. String Directory path. As default is current directory.
  2. String Event separated by a comma.

Examples

$ cat foo.sh
#!/bin/bash
sleep 0.5 && touch foo &
sleep 1.0 && echo lorem ipsum > foo &
sleep 1.5 && unlink foo &
sleep 2.0 && kill -9 $$ &

test-directory.sh .

$ foo.sh
[ >> ] 09:39:55: foo
[ ++ ] 09:39:56: foo
[ xx ] 09:39:56: foo
zsh: killed     foo.sh
