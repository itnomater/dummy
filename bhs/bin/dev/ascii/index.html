Script ascii

Description

Print ASCII character by its code number.

Arguments

  1. String ASCII character (number) to print.

Examples

$ ascii.sh A 0x42 67

A: 0x41 (65)
B: 0x42 (66)
C: 0x43 (67)

You can pass more than one character at once.
