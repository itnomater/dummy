Binaries dev

Helping tools for code development and testing.

  • gendata - Generate random data like numbers, letters, dates, logins, user names, surnames, addresses, etc.
  • ascii - Print ASCII character by its code number.
  • test-bars - Testing for progress bars from bar library.
  • test-colors - Print the content in the specyfic color (Foreground or background).
  • test-colors16 - Test 16-palette terminal colors.
  • test-colors256 - Print 256-color palette.
  • test-tty - Check if UTF-8 characters are available in the current terminal font.
  • test-directory - Monitoring directory changes.
  • test-dialog - Test dialog command abilities.
  • test-whiptail - Test whiptail command abilities.
  • test-cron - Test cron entries.
  • http-request - Send a HTTP request.
