Function opt_get

Description

Print the option value.

See the opt_is function for more information.

Signature

opt_get (

  1. String Option name.

)

Return value

Operation status.

Examples

#!/bin/bash
. ${SHELL_BOOTSTRAP}

lib cmdline "$@"
echo "Option a=$(opt_get 'a')"
echo "Option b=$(opt_get 'b')"
echo "Option c=$(opt_get 'c')"
echo "Option foo=$(opt_get 'foo')"
echo "Option bar=$(opt_get 'bar')"
echo "Option -a=$(opt_get '-a')"
echo "Option -b=$(opt_get '-b')"
echo "Option --foo=$(opt_get '--foo')"
$ /path/to/foo.sh -a 'value a' -b --foo 'value foo' -c -c -c
Option a=value a
Option b=1
Option c=3
Option foo=value foo
Option bar=
Option -a=
Option -b=
Option --foo=

You can pass a blank option value:

./foo.sh -a

Then the option value is set to 1.

When you pass a blank option value many times:

./foo.sh -a -a -a -a

then the option value is set to the number of repetitions.
