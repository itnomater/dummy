Library mysql

Simple library for execute SQL queries.

There is a simple wrapper to mysql command. Credentials to the MySQL server are stored in configuration file (see conf library). By default it is: ~/.mysqlrc

Example structure of the config file is:

[conn]
host          = localhost     #[default: localhost]
charset       = utf8          #[default: utf8]
dbname        = tracehosts    #[default: {empty}]

[cred_ro]       # this user shoud have SELECT privilege only.
user          = user_with_SELECT_privilege_only
pass          = SomePassword

[cred_rw]       # this user shoud have more privilege like DELETE, DROP, TRUNCATE, etc.
user          = user_with_SELECT_and_DELETE_privileges
pass          = SomePassword

Two sets of credentials can be defined. One of them (cred_ro) is only for a user with SELECT privilege, the other (cred_rw) is for a user with more privileges like DELETE, DROP, TRUNCATE, etc.

  • mysql_execute - Execute the SQL query (using credentials for cred_ro).
  • mysql_execute_rw - Execute the SQL query (using credentials for cred_rw).
