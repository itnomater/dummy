Library core

The core library for the Bash Helpers System.

This is the core library of the BHS. It must be loaded to the main script by:

. /path/to/core.inc.sh

Or, use the environment variable SHELL_BOOTSTRAP. See how to install for details.

. ${SHELL_BOOTSTRAP}

And now, you can use helper functions with the BHS main library and load other libraries.
