Function as_root

Description

Check if the script is running as root.

Signature

as_root (

    )

    Return value

    Operation status:

    • 0 As root user.
    • 1 Non-root user.

    Examples

    #!/bin/bash
. ${SHELL_BOOTSTRAP}

echo "Current user: $(whoami)"
as_root \
    && echo 'Script is running as root!' \
    || echo 'Script is running as non-root'
    $ /path/to/foo.sh
foo
Script is running as non-root
