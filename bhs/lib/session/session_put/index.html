Function session_put

Description

Set the session variable.

The variable content is stored in the one file. It allows special character like new line, so that allows to store multiline data instead session_set

Signature

session_put (

  1. String Variable name.
  2. String Variable value.

)

Return value

Operation status.

Examples

#!/bin/bash
. ${SHELL_BOOTSTRAP}

lib session

session_put 'MyVariable' 'foo\nbar\ndoe'
session_get 'MyVariable'
$ /path/to/foo.sh
foo
bar
doe
