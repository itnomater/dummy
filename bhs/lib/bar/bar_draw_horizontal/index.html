Function bar_draw_horizontal

Description

Draw a horizontal progress bar.

Signature

bar_draw_horizontal (

  1. Number Progress in percent.
  2. String Use a frame?
    • noframe|false|0 Do not use a frame.
    • {any_other_value} Use a frame.
  3. String Use a color?
    • nocolor|false|0 Do not use a color.
    • {any_other_value} Use a color.

)

Return value

Operation status.

Examples

#!/bin/bash
. ${SHELL_BOOTSTRAP}

lib bar

for f in `seq 1 10 100`; do 
    printf "%i%% %s\n" \
       $f \
       "$(bar_draw_horizontal $f frame nocolor)"
done
