Function line

Description

Print the full-width separator line.

Signature

line (

  1. Char -c Separator character. [-]
  2. String -f Text color. See colors.
  3. String -b Background color. See colors.

)

Return value

none

Examples

#!/bin/bash
. ${SHELL_BOOTSTRAP}

lib echo3

echo 'Terminal width is 32.'
echo 'Standard line with -'
line
echo 'Standard line with ='
line '='
echo 'Standard line in red color.'
line -f red
$ /path/to/foo.sh
Terminal width is 32.
Standard line with -
--------------------------------
Standard line with =
================================
Standard line in red color.
--------------------------------
