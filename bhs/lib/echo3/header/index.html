Function header

Description

Print the header.

Output format is: {prefix} {header} .

Signature

header (

  1. String -m Header content.
  2. Number -l Header level (indent level).
  3. String -p Header prefix. [>]
  4. String -f Text color. See colors.
  5. String -b Background color. See colors.

)

Return value

none

Examples

#!/bin/bash
. ${SHELL_BOOTSTRAP}

lib echo3

header -m 'Standard heading 1'
header -m 'Standard heading 1' -l 0
header -m 'Standard heading 1' -l 1
header -m 'Standard heading 2' -l 2
header -m 'Standard heading 3' -l 3

header -m 'Heading 1 with #' -p '#'
header -m 'Heading 1 with #' -p '#' -l 0
header -m 'Heading 1 with #' -p '#' -l 1
header -m 'Heading 2 with #' -p '#' -l 2
header -m 'Heading 3 with #' -p '#' -l 3

header -m 'Standard heading 1 with # in red color.' -f red
$ /path/to/foo.sh

> Standard heading 1
> Standard heading 1
>> Standard heading 2
>>> Standard heading 3
