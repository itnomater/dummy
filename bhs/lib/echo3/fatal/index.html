Function fatal

Description

Print the fatal error message in red.

It terminates the script immediately.

Signature

fatal (

  1. String -m Message to print.
  2. String -t The tag.

)

Return value

none

Examples

#!/bin/bash
. ${SHELL_BOOTSTRAP}

lib echo3

fatal -m 'Fatal error occured! Application terminated!'
echo "Fatal break the script so this message won't appear."
$ /path/to/foo.sh
Fatal error occured! Application terminated!
