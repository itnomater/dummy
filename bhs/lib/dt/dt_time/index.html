Function time

Description

Print the time in the specified format.

  1. HH{separator}MM{separator}SS
  2. HHMMSS

Signature

dt_time (

  1. Number -t The UNIX timestamp for the date. If empty it use the current timestamp.
  2. String -s Separator for the date fields [:].
  3. Boolean -n Do not use a separator.

)

Return value

none

Examples

#!/bin/bash
. ${SHELL_BOOTSTRAP}

lib dt

echo 'Current timestamp.'
dt_time
dt_time . '_' # << there is _ character
dt_time . '*'
echo

echo 'Specified timestamp.'
dt_time 89900
dt_time 89900 '_' # << there is _ character
dt_time 89900 '*'
echo

echo 'Command-line style calls.'
dt_time -t 89900 -s '*' 
dt_time -n
$ /path/to/foo.sh
