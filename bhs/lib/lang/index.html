Lang library

The interface for managing the language data in bash scripts.

Separate your code from the language data. No more hardcoded language data in bash scripts. All language data are stored in dedicated language files.

Functions

  • lang_load - Load the language data from the file.
  • lang_get - Print the language variable.
  • lang_set - Set the value of the language variable.

Language file structure

The language data are stored in the extra INI files instead of being hardcoded in a proper bash script. The root directory for the language files is: ${SHELL_LANGDIR}/${LC_MESSAGES}

It is based on the conf library. See configuration file structure.
