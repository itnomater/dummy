Function lang_load

Description

Load the language data from the file.

When you specify only the name of the language file, it will try to load it like below: ${SHELL_LANGDIR}/${LC_MESSAGES}/{name}.lang.ini

Signature

lang_load (

  1. String The language file name. You can pass the absolute path to the file or only the name.
  2. String The section name [DEFAULT].

)

Return value

Operation status.

Examples

#!/bin/bash
. ${SHELL_BOOTSTRAP}

lib lang

lang_load '/path/to/lang' \
    && echo "Language file loaded successfully!" \
    || echo "Failed to load the language file!"
$ /path/to/foo.sh
Language file loaded successfully!

Notice that the above message will appear if the language file /path/to/lang exists and has a valid structure.
