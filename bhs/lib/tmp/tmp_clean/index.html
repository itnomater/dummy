Function tmp_clean

Description

Delete the temporary files/directories.

You should clean all temporary files/directory you have created before the script ends.

Signature

tmp_clean (

  1. String Subdirectory name:
    • {any_value} Specified name.
    • {blank} $(basename $0)-$$

)

Return value

none

Examples

#!/bin/bash
. ${SHELL_BOOTSTRAP}

lib tmp

tmpdir=$(tmp_rootdir)
echo "Current user is: $(whoami)"
echo 'Temporary root directory contains:'
ls -a ${tmpdir}

echo "Create temporary subdirectory: $(tmp_dir)"

echo 'Temporary root directory contains:'
ls -a ${tmpdir}

echo 'Temporary subdirectory deleted.'
tmp_clean

echo 'Temporary root directory contains:'
ls -a ${tmpdir}
