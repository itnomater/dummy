Function tmp_file

Description

Create the temporary file and print its path.

To keep order, it is possible to separate the temporary files into multiple subdirectories using a prefix.

Signature

tmp_file (

  1. String File name:
    • {any_value} Specified name.
    • # or {blank} Random string based on the pattern [0-9a-f]{8}.
  2. String Subdirectory name:
    • {any_value} Specified name.
    • {blank} $(tmp_dir).

)

Return value

none

Examples

#!/bin/bash
. ${SHELL_BOOTSTRAP}

lib tmp
 
tmprootdir=$(tmp_rootdir)
echo "Current user is: $(whoami)"
echo 'Temporary root directory contains:'
ls -a ${tmprootdir}

tmpfile=$(tmp_file 'tmpfile' 'tmpdir') 
tmpdir=$(dirname ${tmpfile})
echo "Create temporary file: ${tmpfile}"

echo "Temporary root directory ${tmprootdir} contains:"
ls -a ${tmprootdir}

echo "Temporary directory ${tmpdir} contains:"
ls -a ${tmpdir}

echo "Temporary file ${tmpfile} deleted."
tmp_clean 'tmpdir'

echo "Temporary root directory ${tmprootdir} contains:"
ls -a ${tmprootdir}

echo "Temporary directory ${tmpdir} contains:"
ls -a ${tmpdir}
