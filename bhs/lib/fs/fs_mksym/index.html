Function fs_mksym

Description

Make a symbolic link to the file.

Signature

fs_mksym (

  1. String Path to the source file.
  2. String Path/name of the symbolic link.

)

Return value

Operation status.

Examples

#!/bin/bash
. ${SHELL_BOOTSTRAP}

lib fs

fpath=/tmp/$(basename $0)
fsymname=$(basename $0)-sym

echo foo bar > ${fpath}
fs_mksym ${fpath} ${fsymname} 

fs_is_reg ${fpath} && printf '%s is a regular file with content:\n' ${fpath} && cat ${fpath}
fs_is_symlink ${fsymname} && printf '%s is a symlink with content:\n' ${fsymname} && cat ${fsymname}

unlink ${fpath}
unlink ${fsymname}
$ /path/to/foo.sh 
/tmp/kk.sh is a regular file with content:
foo bar
kk.sh-sym is a symlink with content:
foo bar
