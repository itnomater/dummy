Function conf_get

Description

Print the config variable value.

Signature

conf_get (

  1. String Configuration variable name in format [section:]key.
  2. String Extra argument for config variable. It replaces any occurrence of ‘%s’ with extra variable from the arguments list, similarly to the printf() function in C.
  3. String .. another argument
  4. String .. another argument
  5. String .. another argument. You can pass many arguments, but the last one denotes a namespace. See the conf_load() function for details. If you use the namespace with conf_load() you need to add ‘@’ character before it. Otherwise just skip it.

)

Return value

Operation status.

Examples

#!/bin/bash
. ${SHELL_BOOTSTRAP}

lib conf

echo '--------------------'
cat '/path/to/configfile'
echo '--------------------'

conf_load '/path/to/configfile' 'section:key' 'DEFAULT' 'EXTRA_PREFIX'
conf_get 'section:key' foo bar doe '@EXTRA_PREFIX'
$ /path/to/foo.sh
--------------------
[section]
key = value 1=%s 2=%s 3=%s
--------------------
value 1=foo 2=bar 3=doe
