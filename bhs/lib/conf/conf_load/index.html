Function conf_load

Description

Load configuration from the file.

See structure of the config file.

Signature

conf_load (

  1. String Path to the configuration file. The name is enough when the configuration file is stored in ${SHELL_CONFDIR}.
  2. String Section name for the variables without specified section. [DEFAULT]
  3. String Namespace of the configuration set. [CONF]

)

Return value

Operation status.

Examples

#!/bin/bash
. ${SHELL_BOOTSTRAP}

lib conf

conf_load '/path/to/config' \
    && echo "Configuration file loaded successfully!" \
    || echo "Failed to load the configuration file!"
$ /path/to/foo.sh
Configuration file loaded successfully!

Notice that the above message will appear if the configuration file /path/to/config exists and it has a valid structure.
