Function conf_clear

Description

Remove (clear) the configuration variable.

Signature

conf_clear (

  1. String Configuration variable name in format [section:]key.
  2. String Prefix for the global environment variable name, see the conf library and conf_load. If you don’t need extra prefixes with conf_load() function then skip this.

)

Return value

Operation status.

Examples

#!/bin/bash
. ${SHELL_BOOTSTRAP}

lib conf

conf_set 'var' 'Variable value'
echo "Variable is: '$(conf_get 'var')'"

conf_clear 'var'
echo "Variable is: '$(conf_get 'var')'"
$ /path/to/foo.sh
Variable is: 'Variable value'
Variable is: ''
