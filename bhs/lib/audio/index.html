Library audio

Helper functions to control the audio.

You can use function as follow:

  • audio_set - Set audio volume level.
  • audio_get - Get audio volume level.
  • audio_up - Increase audio volume level.
  • audio_down - Decrease audio volume level.
  • audio_mute - Mute audio.
  • audio_unmute - Unmute audio.
  • audio_toggle - Toggle audio mute state.
  • audio_check - Check audio mute state.
