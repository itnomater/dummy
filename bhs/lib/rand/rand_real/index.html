Function rand_real

Description

Generate a random real number.

Signature

rand_real (

  1. Number Number digits of integer part [4].
  2. Number Number digits of decimal part [2].
  3. Number Probability of use negative value [0].

)

Return value

Operation status.

Examples

#!/bin/bash
. ${SHELL_BOOTSTRAP}

lib rand

echo 'Small positive numbers with large decimal part:'
for i in {1..10}; do
    echo $(rand_real 2 100)
done

echo 'Large positive numbers with small decimal part:'
for i in {1..10}; do
    echo $(rand_real 100 2)
done

echo 'Numbers with 50% probability to negative value:'
for i in {1..10}; do
    echo $(rand_real 4 2 50)
done
$ /path/to/foo.sh
