Function rand_probe

Description

Get the truth with a certain probability.

Signature

rand_probe (

  1. Number Probablity value in range (0-100). As default is 50.

)

Return value

Operation status.

Examples

#!/bin/bash
. ${SHELL_BOOTSTRAP}

lib rand

number=0
for p in 1 9 20 50 75 99 100; do
    number=0
    for i in {1..100}; do
        rand_probe $p && let number++
    done
    printf "Test probability with %i%% is: %i\n" \
           ${p} \
           ${number}
done
